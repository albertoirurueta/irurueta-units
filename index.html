About

Library to manage different measurement units

Set up dependencies

To use this library add the following dependency to your pom.xml file:

<project>
    ...
    <dependencies>
        ...
        <dependency>
            <groupId>com.irurueta</groupId>
            <artifactId>irurueta-units</artifactId>
            <version>1.0-SNAPSHOT</version>
        <dependency>
        ...
    </dependencies>
    ...
</project>
